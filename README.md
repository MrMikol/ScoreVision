# ScoreVision 🎵

A music sight reading trainer app built with React Native + Expo.

## Getting Started

### Prerequisites
- Node.js installed
- Expo Go app installed on your phone
- Phone and PC on the same WiFi network

### Run the app
```bash
npx expo start
```
Then scan the QR code with your iPhone camera.

## Game Modes
- 🎯 **Challenge** — Guess a set number of notes (10, 20, or 30)
- ⏱ **Time Attack** — Answer as many as possible before time runs out (60s, 90s, 120s)
- ♾ **Endless** — Keep going until you stop

## Difficulty Levels
- **Easy** — Treble clef only, notes within the staff, no accidentals
- **Medium** — Treble & Bass clef, includes ledger lines, no accidentals
- **Hard** — Both clefs, full range, sharps & flats included

## Tech Stack
- React Native
- Expo
- React Navigation
- React Native SVG
